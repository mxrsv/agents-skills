# Layout — composition with gravity, rhythm, intentional asymmetry

- **Asymmetry with intent**: hero = dominant headline left + focal artifact right
  (not a centered headline on a void — that "looks like every other hero").
- **One dominant element** per view; the rest recedes. If everything is bold,
  nothing is. Composition should have a clear first/second/third read.
- **Rhythm connects sections**: a consistent gutter (`clamp()`), a section-padding
  scale, hairline separators between sections so they feel like one document.
- **Engineered framing**: corner ticks / registration marks / hairline rules give a
  precise, drafted feel (vs floating cards on a void).
- **Every section earns its place** — no filler, no repeating the same artifact
  twice, no "Feature One / lorem / zeros".
- **Layered depth**: background → blocks → detail → interaction → motion. Build the
  page in those layers, not one flat plane.
- **Responsive is its own design**: grid → 1 column, horizontal-scroll → vertical
  stack, oversized type scales down — never a squished desktop. Hover/focus states
  are part of the design.
- **Full-page background**: a per-hero radial mask concentrates texture at the top;
  for a long page swap to a horizontal vignette (full-height) or pin the field, so
  texture stays consistent down every section.
