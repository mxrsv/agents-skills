# Gallery — reference library (append-only)

Drop references here as they come in. The point is NOT to clone a style — it's to
read the user's **taste fingerprint** (the through-line across different-looking
refs) and the **system** behind each. Different styles are welcome; the common
thread is the signal. Distil findings into `frontend-taste-profile.md` (memory).

## How to add an entry

- Save a screenshot to `screenshots/` (e.g. `screenshots/2026-06-26-sitename.png`).
- Add a row. The **"what's good"** + **tags** are the part that matters — that's the
  system to extract, not the URL.
